**数据集名称**：TCM-Diagnosis  
**评估维度**：诊断推理（Diagnostic Reasoning）

**数据集描述**：TCM-Diagnosis 评估模型在中医临床场景下的辨证推理能力。每条样本以术语规范化的临床表现（症状与体征）为输入，要求模型输出疾病名称、证名及证素（病位、病性）等结构化诊断结论。数据集涵盖内科、外科、妇科、儿科各 50 例，共 200 例。

**数据集来源**：国家“十三五”规划教材——《中医内科学》《中医外科学》《中医妇科学》《中医儿科学》（新世纪第四版）。

**构建方式**：以教材病例的规范化临床表现为输入；由持证中医执业医师标注疾病名称、证型及证素作为标准答案；经多轮专家审核。

**元数据**：

```
question：规范化临床表现（输入）
answer：疾病名称、病位证素、病性证素、证名（结构化）
source：样本溯源
```

**评估指标**：BLEU、ROUGE-L 与 BERTScore 的算术平均
