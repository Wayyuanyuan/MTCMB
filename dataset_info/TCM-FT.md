**数据集名称**：TCM-FT  
**评估维度**：知识问答（Knowledge Question Answering）

**数据集描述**：TCM-FT 包含 100 道开放式专家级问答题，用于评估大模型以精确、专业的方式阐述中医概念的能力。内容覆盖中医基础理论、诊断、方药、针灸及临床各科等。

**数据集来源**：《中医学问答题库》（胡熙明主编，中医古籍出版社，1988）。

**构建方式**：从题库随机抽样；使用 DeepSeek-R1 对题目进行语义改写，并提取要点作为参考答案（`points` 字段）；经专家交叉验证确保问答一致性。

**元数据**：

```
question：题目（输入）
points：要点式标准答案（评测对齐字段；完整长答案见 data 中 answer）
source：样本溯源
```

**评估指标**：BERTScore（模型输出与 `points` 要点的语义对齐，按实例平均后缩放至百分制）

**示例**：

```
 {
        "id": 1,
        "question": "何谓血瘀?血瘀是如何形成的?",
        "points": "- 血瘀指血液的循行迟缓和流行失畅的病理状态\n- 血瘀形成原因包括气滞而血行闭阻\n- 气虚而血运迟缓\n- 痰浊阻络而血运失畅\n- 阳虚或寒邪阻滞血脉、血寒而凝\n- 邪热入血，煎熬津液，津液干涸而致瘀"
    }
```
