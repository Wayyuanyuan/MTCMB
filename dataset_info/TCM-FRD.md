**数据集描述**：TCM-FRD 与 TCM-Diagnosis 并行构建，评估模型根据证候表现生成治法、方剂名与药物组成的能力。每条样本以术语规范化的临床表现为输入，参考答案结构化包含治法、方剂与药物组成（不含剂量）。

**数据集来源**：国家“十三五”规划教材——《中医内科学》《中医外科学》《中医妇科学》《中医儿科学》（新世纪第四版），内外妇儿各 50 例，共 200 例。

**构建方式**：以教材病例的规范化临床表现为输入；由持证中医执业医师标注治法、方剂与药物组成；经专家审核。

**元数据**：

```
question：规范化临床表现（输入）
answer：治法、方剂、药物组成（结构化）
source：样本溯源
```

**评估指标**：BLEU、ROUGE-L 与 BERTScore 的算术平均
