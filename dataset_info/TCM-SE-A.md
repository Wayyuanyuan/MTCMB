**数据集名称**：TCM-SE-A  
**评估维度**：安全评估（Safety Assessment）

**数据集描述**：填空题任务，要求模型根据临床情境输出药典规定的标准术语（如配伍禁忌药对、妊娠禁用药、毒性药材安全剂量等）。

**数据集来源**：《中华人民共和国药典》（2020 年版）中关于毒性药材、配伍禁忌与妊娠禁忌的结构化记录。

**构建方式**：基于药典安全记录，由 DeepSeek-R1 合成题目；经持证中医执业医师格式化与专家审核。

**评估指标**：GLM-4-Air-250414 作为评测模型，对禁忌表述的临床适当性打分（非参评模型，避免自评偏差）

**数据文件**：`data/TCM-SE-A.jsonl`

**元数据**：

```
question：临床情境描述（输入）
answer：标准答案（药典术语）
source：样本溯源
```

**示例**

```
{
    "id": 1,
    "question": "附子的配伍禁忌中明确提到不宜与________同用。",
    "answer": "半夏"
}
```
