**数据集描述**：TCM-SE-A 与 TCM-SE-B 评估大模型对中药及针灸安全知识的掌握，涵盖毒性中药、配伍禁忌（十八反、十九畏等）与妊娠禁忌等。TCM-SE-A 为填空题（50 题），TCM-SE-B 为选择题（50 题）；均以临床情境为输入。

**数据集来源**：《中华人民共和国药典》（2020 年版）中关于毒性药材、配伍禁忌与妊娠禁忌的结构化记录。

**构建方式**：基于药典安全记录，由 DeepSeek-R1 合成题目；经持证中医执业医师格式化与专家审核。SE-A 要求输出药典规定的标准术语；SE-B 提供候选选项与标准答案。

**元数据**：

```
question：临床情境描述（输入）
options：选项（仅 SE-B）
answer：标准答案（药典术语或选项标签）
source：样本溯源
```

**评估指标**：

- **TCM-SE-A**：GLM-4-Air-250414 作为评测模型，对禁忌表述的临床适当性打分（非参评模型，避免自评偏差）
- **TCM-SE-B**：准确率（Accuracy）

**数据文件**：`data/11.TCM_DC_A.jsonl`（SE-A）、`data/12.TCM_DC_B.jsonl`（SE-B）

**示例**

```
 # 选择题（TCM-SE-B）
    "id": 1,
        "question": "孕妇禁用的中药材是以下哪种？",
        "options": "A. 胆南星 B. 土鳖虫 C. 艾叶 D. 半夏",
        "answer": "B"
    }
 # 填空题（TCM-SE-A）
    {
        "id": 1,
        "question": "附子的配伍禁忌中明确提到不宜与________同用。",
        "answer": "半夏"
    }
    
```
