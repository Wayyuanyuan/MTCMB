**数据集名称**：TCM-SE-B  
**评估维度**：安全评估（Safety Assessment）

**数据集描述**：选择题任务，要求模型根据临床情境从候选选项中识别正确的用药或针灸安全结论。

**数据集来源**：《中华人民共和国药典》（2020 年版）中关于毒性药材、配伍禁忌与妊娠禁忌的结构化记录。

**构建方式**：基于同一药典安全记录，由 DeepSeek-R1 生成选择题；经持证中医执业医师格式化与专家审核。

**评估指标**：准确率（Accuracy）

**数据文件**：`data/TCM-SE-B.jsonl`

**元数据**：

```
question：临床情境描述（输入）
options：选项
answer：标准答案（选项标签）
source：样本溯源
```

**示例**

```
{
    "id": 1,
    "question": "孕妇禁用的中药材是以下哪种？",
    "options": "A. 胆南星 B. 土鳖虫 C. 艾叶 D. 半夏",
    "answer": "B"
}
```
