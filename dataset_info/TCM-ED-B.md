**数据集名称**：TCM-ED-B  
**评估维度**：知识问答（Knowledge Question Answering）

**数据集描述**：TCM-ED-B 包含 4,800 道单项选择题，源自 8 套完整的国家中医执业医师资格考试标准化模拟试卷（每套 600 题），用于评估大模型在 TCM 领域的广博知识覆盖。

**数据集来源**：国家中医执业医师资格考试官方题库（才识教育科技标准化题库）。

**构建方式**：从 8 套已发布标准答案的模拟卷中随机整卷纳入；经专家 panel 验证学术权威性与临床相关性。

**元数据**：

```
question：题干
options：选项
answer：标准答案（单字母 A–E）
source：样本溯源
```

**评估指标**: 准确率（Accuracy）

**示例数据**

```
 {
        "id": 1,
        "question": "1.足太阴脾经与手少阴心经的交接位置是",
        "options": [
            "A.胸中",
            "B.肺中",
            "C.心中",
            "D.目内眦",
            "E.足大趾"
        ],
        "answer": "C"
    }
```
